AI IMPAIRMENT CHECKS

1. Stability Check
  Instructions:
    User holds phone to chest or places in pocket.
    Collect accelerometer + gyroscope data for 15 seconds.
    Compute:
      Sway amplitude variance
      Drift frequency
      Micro-oscillation irregularity
    Compare to: UserProfile.baselineStabilityVariance
    Return:
      stabilityDeltaPercent
      impairmentContributionScore

2. Reaction Check
  20-second dual-task test:
    Tap screen when color changes
    Randomized interval
    Optional simple memory task
  Measure:
    Average reaction time
    Miss rate
    Variance
  Compare to: baselineReactionTime
  Return:
    reactionSlowdownPercent
    impairmentContributionScore

3. Focus Check (Camera)
  Use front camera for 10 seconds.
  Track:
    Facial landmarks
    Eye tracking smoothness
    Head sway drift
  Measure:
    Smooth pursuit stability
    Tracking jitter variance
    Drift deviation
  Compare to: baselineFocusScore
  Return:
    focusDeltaPercent
    impairmentContributionScore

IMPAIRMENT RISK CALCULATION
FinalRiskScore = weighted fusion of:
  Estimated BAC midpoint
  Stability impairment score
  Reaction impairment score
  Focus impairment score

Use weighted model:
  If only 1 test taken: ConfidenceScore = low (40–60%)
  If 2 tests: ConfidenceScore = medium (60–80%)
  If 3 tests: ConfidenceScore = high (80–95%)

Output:
  ImpairmentRiskLevel:
    Low
    Elevated
    High
    Severe

NEVER output:
  "Legally drunk"
  "Safe to drive"

Always frame as:
  "Performance-based impairment risk"
